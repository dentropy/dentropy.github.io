  * User on server
  * Channel
  * Server
  * User on channel
* Messages per period of time
	* Time Periods
  	* Day of week
  	* Time of day
  * What to analyze
    * User across servers
    * User on server
    * Channel
    * User on channel
    * User on server
* Characters Per Message
  * Over Time
  * Average for channel
  * Average for User
  * Average per period of time in channel

* Reactions / Emoji's
	* Most per message
  * Popularity used by specific user
  * Populatirity within speicifc channel
