## Query Name

guild_authors

## SQL Query


``` sql

select
	distinct(authors_t.id) as author_guild_id,
	authors_t.author_name,
	author_name,
	nickname,
	authors_t.id as guild_id
from
	authors_t
join guilds_t on authors_t.guild_id = guilds_t.id
where
guild_id in (  (select id from guilds_t limit 1)  );
