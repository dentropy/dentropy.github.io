## Install

``` bash
cd  $VAULT
cd .obsidian/plugins
git clone https://github.com/LordGrimmauld/aw-watcher-obsidian.git
cd aw-watcher-obsidian
npm install
npm run build
```

Go to `settings` -> `community plugins` and enable ActivityWatch
