## Query Name

guild_bots

## SQL Query

``` sql

select 
	guilds_t.guild_name,
	author_name,
	is_bot,
	guild_id,
	author_id,
	authors_t.id
from
	authors_t 
join guilds_t on guilds_t.id = authors_t.guild_id
where 
	is_bot = 'T'
	and guild_id in (  (select id from guilds_t limit 1)  );
```
