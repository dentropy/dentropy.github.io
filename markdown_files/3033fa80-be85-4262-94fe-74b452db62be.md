# /etc/passwd and /etc/shadow
*   /etc/passwd
    *   Username
    *   Password
    *   UID
    *   GID
    *   Comment
    *   Directory
    *   Shell
*   /etc/shadow
    *   Login Name
    *   Encrypted Password
    *   Day number since unix time started password change
    *   Days before password may be changed
    *   Days after password expires and account is disabled
    *   Day that account is disabled
    *   Future Use
