title: What is the number of channels within a specific discord guild?
---
## Query Name

guild_channels_count

## SQL Query


``` sql

select
	count( distinct(channels_t.id) ) as channel_count
from
	channels_t
where
	guild_id = (select id from guilds_t limit 1);

```
