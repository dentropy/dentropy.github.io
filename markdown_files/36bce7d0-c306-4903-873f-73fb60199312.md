## Schema

* Message ID
  * "message_id"
* Message Contents
  * "content"
* User
  * "author"
    ``` json
    {
        "author" : {
            "id" : "432981598818903585",
            "name" : "dentropy",
            "discriminator" : "8658",
            "nickname" : "Paul",
            "color" : "#2ECC71",
            "isBot" : false,
            "avatarUrl" : "https://cdn.discordapp.com/avatars/............"
        }
    }
    ```
* Channel
  * "channel_id"
* Guild
  * "guild_id"
* User Mentions
  * "mentions"
* Replied
* Reactions
  * "reactions"
