## Custom Hotkeys and Settings

* Input Timestamp
	* Ctrl + Shift + R
* Open Daily Note
	* Ctrl + Shift + I
* Move Note
	* Ctrl + Shift + M
* Pin Note
	* Ctrl + Shift + P
* Linked Mentions
* No line length
