## Tutorial for Linux

1. Download tar.xz release from [Releases Â· ventoy/Ventoy](https://github.com/ventoy/Ventoy/releases)
2. Extract and run
	``` bash
	cd ~/Downloads
	tar -xvf ventory*
	cd ventoy-TAB
	./VentoyGUI.x86_64
	```
3. Select drive within GUI and burn
4. Copy ISOs to drive, yup that's literally it
