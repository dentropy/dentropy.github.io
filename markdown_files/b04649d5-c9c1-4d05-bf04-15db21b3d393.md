## Install Instructions

* [Installation - WireGuard](https://www.wireguard.com/install/)

``` bash
sudo su
sudo apt-get -y wireguard
sudo apt-get -y install resolvconf
# Put .conf file in /etc/wireguard
wg-quick up `filename of .conf file` 
```
