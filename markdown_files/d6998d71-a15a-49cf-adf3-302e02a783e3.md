#### [What is a static site generator?](../What is a static site generator?)

A static site generator takes content, traditionally [Markdown Files](../ceb61472-539e-4d96-8db5-9168bf51b6e9) but can be word documents and other various formats, then produces a directory of HTML and other various files that can be hosted on a HTTP server like [nginx](../2c37e79d-9050-4762-8cc6-42f9060bd348) without any database or other fancy software.
