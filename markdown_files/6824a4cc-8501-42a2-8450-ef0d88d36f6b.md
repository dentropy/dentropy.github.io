## Example
``` bash

# Sync a directory
mega sync $LOCAL_PATH $MEGA_PATH
mega-sync $LOCAL_PATH $MEGA_PATH
mega-sync /home/dentropy/umbrel/data/storage/downloads/audiobooks /BooksAndDocuments/Audiobooks

# Check SYnc Status
mega-cmd
sync
transfers --help
transfers --show-syncs

```
