## GraphQL Query

``` json
query MyQuery {
  allChannelsTs(condition: {guildId: "621022643344113684"}) {
    edges {
      node {
        id
        channelName
        channelType
        guildsTByGuildId {
          id
        }
        messagesTsByChannelId {
          aggregates {
            distinctCount {
              author
            }
          }
        }
      }
    }
  }
}
```
