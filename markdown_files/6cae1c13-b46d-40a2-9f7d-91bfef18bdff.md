## Commands
 
``` cypher
:server status
:server connect
:sysinfo
:clear
:history
:play start
:help
  :help keys
  :help cypher
  :help Schema
  :help with
  :help match
  :help return
  :help create-constraint-on
  :help create-index-on
  :help load-csv
```

## Running Scripts

``` bash
cypher-shell -f yourscriptname
cat neo_4.cypher | cypher-shell
sudo cypher-shell arguments_as_needed < neo_4.cypher
```
